## How to run

```bash
# Clone the Repository
$ git clone https://github.com/CarlosEduAC/node-with-typescript
```

```bash
# Go to the server folder
$ cd node-with-typescript

# Install the dependencies
$ yarn install

# Run the application
$ yarn dev:server
```
Access the API: http://localhost:3333/
